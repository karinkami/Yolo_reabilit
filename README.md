# Yolo_reabilit
YOLO Reabilit — веб-система реабилитации в больнице. Врач назначает упражнения; пациент выполняет их перед веб-камерой и получает подсказки в реальном времени. Система распознаёт позу, считает углы в суставах и по правилам засчитывает повторы; результаты сохраняются в статистике для врача и пациента.
