# YOLO Reabilit

Веб-система реабилитации: врач назначает упражнения, пациент выполняет их дома перед веб-камерой с подсказками в реальном времени, результаты сохраняются для врача и пациента.

## Идея

Пациент выполняет назначенные врачом упражнения; камера передаёт видеопоток на сервер. Система распознаёт позу, считает повторы и показывает подсказки. Врач видит историю тренировок и прогресс.

Сочетание **оценки позы** (нейросеть находит суставы в кадре) и **правил реабилитации** (пороги, фазы, тексты — без отдельного обучения под каждого пациента).

**Роли:** врач — назначения и статистика; пациент — тренировка и своя статистика.

---

## Основное

- Упраднения на руки, локоть, приседания, дыхание 
- Съёмка **боком** к камере: рабочая рука или нога ближе к объективу
- Счётчик повторов, подсказки на русском, сохранение сессии в PostgreSQL
- React-интерфейс: кабинет врача, страница тренировки, графики

**Стек:** Python, Flask, PostgreSQL, React, YOLO pose (Ultralytics), OpenCV.

## Быстрый старт

Полная пошаговая инструкция — **[docs/ИНСТРУКЦИЯ_ПОДКЛЮЧЕНИЯ.md](docs/ИНСТРУКЦИЯ_ПОДКЛЮЧЕНИЯ.md)**.

Кратко (Windows, из корня проекта):

```powershell
python -m venv yolo_env
.\yolo_env\Scripts\Activate.ps1
pip install -r requirements.txt
```

1. **PostgreSQL** — роль и база `yolo_rehab`, файл `appsettings.json` (из `appsettings.example.json`).
2. **Фронтенд** — `cd frontend` → `npm install` → `npm run build`.
3. **Запуск** — `python run.py` → [http://127.0.0.1:5000](http://127.0.0.1:5000).


Без `appsettings.json` приложение не стартует. Без сборки `frontend` — ошибка **503**.
