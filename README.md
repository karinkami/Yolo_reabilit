# YOLO Reabilit

Веб-приложение для реабилитации с контролем упражнений через веб-камеру. Врач назначает программу и следит за прогрессом; пациент выполняет упражнения дома и получает подсказки в реальном времени.

## Что это

**YOLO Reabilit** помогает проводить ЛФК под контролем камеры без сложного оборудования.

- врач создаёт назначения (упражнение, сторона, число повторов);
- пациент тренируется на странице с видео и подсказками «Сейчас» / «Советы»;
- система считает повторы и сохраняет результат в PostgreSQL;
- врач и пациент смотрят историю и графики.

Технически: **YOLO pose** находит суставы в кадре, **правила в коде** решают, засчитан ли повтор и что показать пользователю.

## Быстрый старт

Подробные шаги — в [инструкции](docs/ИНСТРУКЦИЯ_ПОДКЛЮЧЕНИЯ.md).

```powershell
python -m venv yolo_env
.\yolo_env\Scripts\Activate.ps1
pip install -r requirements.txt
```

Скопируйте `appsettings.example.json` → `appsettings.json`, настройте PostgreSQL, затем:

```powershell
cd frontend
npm install
npm run build
cd ..
python run.py
```

Сайт: [http://127.0.0.1:5000](http://127.0.0.1:5000) · демо-врач: `doctor@clinic.local` / `doctor123` · тренировка: `/patient/rehab/`
